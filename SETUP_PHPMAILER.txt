# SETUP PHPMailer NOW - 3 STEPS

## Step 1: Install PHPMailer

**Choose ONE option:**

### Option A - Easiest (Windows)
```
1. Double-click: install-phpmailer.bat
2. Wait for it to complete
3. Press Enter to close
```

### Option B - Manual (All Platforms)
```
1. Open PowerShell in project folder
2. Type: composer install
3. Press Enter
```

## Step 2: Verify Installation

After installation, you should see:

## Common Issues

| Problem | Solution |
|---------|----------|
| "Composer not found" | Download from https://getcomposer.org/ |
| "vendor/ folder missing" | Run `composer install` again |
| "Email not arriving" | Check spam folder, verify internet connection |
| "Authentication failed" | SMTP credentials are correct in code |
